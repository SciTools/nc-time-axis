Installation
============

Minimum Requirements
--------------------

* `cftime`_ >= 1.5
* `matplotlib`_ >= 3.8.0
* `numpy`_ >= 1.25
* `python`_ >= 3.11

Installation Methods
--------------------

``nc-time-axis`` can be installed through `conda`_::

    conda install -c conda-forge nc-time-axis

or through `pip`_::

    pip install nc-time-axis
